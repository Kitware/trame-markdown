## alter

Markup is similar to fenced code blocks. Valid container types are `tip`, `info`, `warning`, `danger` and `details`.

::: tip
You have got it.
:::

::: info
You have new mail.
:::

::: warning
You have new mail.
:::

::: danger
Staying up all night is bad for health.
:::
